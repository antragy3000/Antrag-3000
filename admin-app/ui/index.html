Antrag 3000 – Admin

Anmelden

Wähle dein Zugangs-Paket (.a3kpaket) – dasselbe wie in der Nutzer-App – und gib den aktuellen 6-stelligen Code aus deiner Authenticator-App ein.

Förderer aktivieren

Einmalig eine Förderer-Signaturstelle erstellen, dann pro Förderer eine Aktivierungs-Datei ausstellen. Damit signiert die Förderer-App ihre Exporte – jede Förderung ist so fälschungssicher diesem Förderer zuordenbar.

Sammler-Vorschläge

Vom Sammler vorgeschlagene neue oder geänderte Förderungen. Freigeben übernimmt sie in den verteilten Katalog, verwerfen lehnt sie ab.

Förder-Katalog bearbeiten

Alle Förderungen im verteilten Katalog. Bearbeiten ändert einen Eintrag direkt; gespeicherte Änderungen werden sofort an alle Team-Geräte verteilt (Stand wird auf heute gesetzt).

Regelmäßige Prüfung

Jede Förderung soll alle 3 Monate einmal von Hand geprüft werden (stimmen Angaben, Fristen, Webseite noch?). Überfällige stehen oben. ✓ geprüft setzt das Prüfdatum auf heute – das löst kein Update bei den Nutzer:innen aus.

Gemeldete Probleme

Was Nutzer:innen an falschen/veralteten Förderungen gemeldet haben.

Vom Team geteilte Förderer

Selbst recherchierte Förderer (öffentliche Felder), die das Team geteilt hat.

Förder-Katalog hochladen

Wähle die foerderungen.json. Danach zwei Möglichkeiten: Ergänzen fügt die Förderungen aus der Datei zum bestehenden Katalog hinzu – gleiche id wird aktualisiert, alles andere bleibt erhalten (empfohlen, z. B. für neu recherchierte Förderungen). Ersetzen wirft den gesamten bisherigen Katalog weg und nimmt nur die Datei.
